You have array of integers.

Find 2 numbers in array that will sum to target and return their indices.

e.g.

Numbers: [2,7,11,15]
Target: 9
Out: [0, 1]

## How to run tests

### Go
```bash
go test ./go/...
```

### Python
```bash
python3 py/test_index.py
```

### Lua
```bash
lua lua/index_test.lua
```

### JavaScript (Bun)
```bash
cd js && bun test
```
